Parágrafos e quebras de linha

podemos escrever textos ou parágrafos usando <p> Também podemos quebrar linhas com a tag <br>
É também podemos combinar estas duas tags para fazer o exemplos abaixo, a tag <p> para criar o parágrafo, criando a base, e a tag <br> para quebrar as linhas que queramos.

exemplo:
Come as you are, as you were, as I want you to be

As a friend, as a friend, as an old enemy

Take your time, hurry up, the choice is yours, don't be late

Take a rest as a friend as an old memoria

podemos usar esta tag <hr>

para inserir uma linha horizontal na página da web. Esta linha pode ser usada para separar o conteúdo, criando uma divisão visual clara entre as partes do documento.
